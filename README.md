# 2zcory Garden

Personal web application for `2zcory`.

## Purpose

- Present a living personal profile, resume, projects, writing, and knowledge garden
- Keep public identity and proof of work aligned with one private source of truth
- Let Context OS support content development and publishing over time

## Repository Pair

- Source repo: `~/wp/2zcory-garden`
- Context repo: `~/wp/2zcory-garden-ctx`

## Initial Scope

- Public surfaces: Home, About, Resume, Projects, Garden, Writing, Now, Contact
- Private studio: Dashboard, Knowledge, Resume Studio, Project Studio, Publishing, Review, Settings
- First implementation focus: establish product definition, IA, content model, and MVP scope in the context repo before UI design and code implementation
